### №1

**Задание**

- Замыкание-калькулятор, накапливающее результат и поддерживающее 4 арифметичесике операции:

``` Python
>>> calc = make_calc("*", initial=1)
>>> calc(5)

>>> calc(2)
10
```

Создана функция make_calc, которая принимает два параметра: operation - арифметическое действие и initial - начальное значение, внутри функции создана переменная result, которая хранит результат вычислений. Создана внутренняя функция calculator, которая Принимает число num, с помощью nonlocal result получает доступ к изменению переменной result из внешней функции. В зависимости от выбранной операции выполняет вычисление и возвращает обновленный результат

### №2

**Задание**

- Декоратор, который будет запускать функцию указанное число раз с указанными параметрами и возвращать последвательность результатов.


**Результат**

<img width="966" height="34" alt="image" src="https://github.com/user-attachments/assets/bbf2255f-31d9-47cd-9978-9486dfe8f51d" />

**Описание проделанной работы**

Внешняя функция repeat принимает параметр times — количество запусков декорируемой функции, возвращает функцию decorator, которая и будет декорировать целевую функцию. Функция decorator принимает декорируемую функцию func и возвращает функцию wrapper. Принимает аргументы, переданные при вызове (*args), создаёт пустой список results для хранения результатов в цикле for i in range(times) заданное количество раз вызывается исходная функция func с переданными аргументами и добавляется полученный результат в список results, возвращает список всех результатов.
