**Задание**

Реализуйте приложение с GUI (приложения-игры допускается делать с использованием TUI-пакетов) по своему варианту.
- Приложение должно быть написано с применением ОО парадигмы
- Исключительные ситуации должны обрабатываться с использованием собственных исключений

**Название приложения**

Система учета задач (ToDo)

**Описание**

**Инструкции по запуску**

Проверить версию питона

```
python --version
```

Установить библиотеку appjar 

```
pip install appJar
```

Запустить файл

```
python3 main.py
```

**Краткую справку**

Кнопки управления

Добавить - создает новую задачу
Выполнено - меняет статус выбранной задачи
Удалить - удаляет выбранную задачу
Сменить вид - переключает режим отображения

Статусы задач

[✓] - Выполнено 
[○] - Не выполнено

Режим отображения 

Обычный вид 

1. [✓] Купить хлеб
2. [✓] Сделать дз

Вид с датой

1. [✓] Купить хлеб (14.05)
2. [✓] Сделать дз (14.05)
