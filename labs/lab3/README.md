### №1

**Общее задание**

Написать две функции для решения задач своего варианта - с использованием рекурсии и без.

**Задание**
- Функция для подсчёта числа элементов в списках, включая вложенные списки:
  
  ```python
  >>> count([])
  0
  >>> count([1, 2, 3])
  3
  >>> count(["x", "y", ["z"]])
  4
  >>> count([1, 2, [3, 4, [5]]])
  7
  ```
**Результат**

<img width="992" height="157" alt="image" src="https://github.com/user-attachments/assets/1afb4f8a-9650-4d1a-8458-92ca8e76ae7f" />

**Описание проделанной работы**

С рекурсией:

Обхожу каждый элемент списка и увеличиваю счетчик на 1 если это обычный элемент, если элемент оказывается списком, функция вызывает саму себя для этого вложенного списка, результат вызова добавляется к общему счетчику.

Без рекурсии:

Создаётся пременная (process) в которой будет содержаться список и создается счетчик. Пока есть список будет выполняться: извлекается первый список из очереди далее он удаляется, происхоит проверка если элемент является списком добавляется в конец и прибавляется 1 в счетчик, если обычный элемент то просто добавляется 1.

**Задание**

- Функция для расчёта $$x_i = \frac{(i-1)x_{i-1}}{3} + \frac{(i-2)x_{i-2}}{4},\quad x_1 = 1,\quad x_2 = -\frac{1}{8}$$
  
**Результат**

**Описание проделанной работы**

С рекурсией: 

Если i будет равно 1 вернет 1, если 2 вернет -1/8. Для чисел больше 2 будет вызывать саму себя.

Без рекурсии: 

Для 1 и 2 также как в рекурсии. Для чисел 2 больше двух вычисляет новое значение по формуле, обновляет переменные для следующей итерации и так до нужного i.
